Use menu to higlight specific compounds.
Elements must be separated by comma and start with upper case letter. Once typed in, press "Enter".

To view deatils of non-highlighted compounds enable the second "Hower" tool.

Use "Reset" tool to normalize axis values after zooming or switching axis.

Use selection tools ("Lasso", "Tap") to see more details in the table below.
Hold down shift to append to selection. Pres Esc to cancel selection.
Non-highlighted compounds cannot be selected.

Press "Save table" button to export the table as a .csv file.

Use "Save" tool button to export the plot as a .png image.
