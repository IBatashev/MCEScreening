version 1.3

This guide describes all interactive elements and controls. Descriptions are grouped in several sections according to element utility. For the physical meaning behind each parameter see info.pdf.

SETTINGS

Database file: select data source for the plot. These files must be *.csv located in the same directory as MCES.exe. Currently, two files are available:
- Full_DB - contains all successfully calculated materials from the Materials project database that suit initial conditions
- Top_candidates - a subset of Full_DB that comes from applying additional sieving criteria.

X-Axis: chose x-axis for the plot.

Y-Axis: chose y-axis for the plot.

CONDITIONS

The final list of conditions follows AND logic, so only database entries that satisfy all chosen conditions simultaneously will be highlighted.

For condition boxes where text can be typed in, to submit your input press 'Enter' on the keyboard or mouse click outside the textbox.

Clicking on the green "Export Data" button will produce a data_result.csv file. This file contains detailed information for all compounds that fit current choice of conditions.

Database ID: Label used by Materials Project. Input any number of ids separated by a comma. Case sensitive. Internal logic is OR.

    Example input: mp-778, mp-238

Compound: Chemical formula of the material. Input one or several separated by a comma. Case sensitive. Internal logic is OR. Must be an exact match to the compound(s) in the database (e.g FeB is not the same as BFe or Fe2B2) so it may be necessary to try some permutations of the chemical formula or to use the "Contains Element(s)" condition instead.

    Example input: Fe2P, MnPd

Contains Element(s): All elements that must be present in the composition. Internal logic is AND. Input one or several separated by a comma. Case sensitive. The order of the elements is irrelevant.

    Example input: Fe, Mn, P

Does not contain Element(s): All elements that are not allowed in the composition. Internal logic is AND. Input one or several separated by a comma. Case sensitive. The order of the elements is irrelevant.

    Example input: O, B

Maximum allowed price (EUR/kg): Price of the material that must not be exceeded. Input a number. By default, the application will start with the maximum price found among all the database entries.

    Example input: 2000

Lattice type(s): Seven checkboxes, one for each symmetry type. Select one or more to limit the search only to selected symmetries. Internal logic is OR (e.g., by checking 'cubic' and 'hexagonal' only systems with these two lattice types will be selected). No selection is the same as selecting all options.

Magneto Elastic Full Uniaxial: Minimum allowed value of the magneto elastic uniaxial parameter. Set desired value with a slider.

Magneto Elastic Volumetric: Minimum allowed value of the magneto elastic volumetric parameter. Set desired value with a slider.

Internal Field: Minimum allowed value of internal field parameter. Set desired value with a slider.

Applied Field Response: Minimum allowed value of applied field response parameter. Set desired value with a slider.

Magnetically active atoms %: Minimum allowed percentage of magnetically active atoms in the composition. Set desired value with a slider.

Rare Elements allowed %: Maximum allowed percentage of the rare earth elements in the composition. Set desired value with a slider.

WARNING: Not all compounds have data for volumetric magneto elastic parameter or applied field response. When this information is unavailable, values in question default to 0.0. Therefore, any filtering with these parameters should be done with care.

PLOT

The plot area is populated by database items.

Above the plot, the number of items that satisfy currently selected conditions is displayed.

Orange data points represent items that satisfy currently selected conditions.

Black data points represent items that do not satisfy currently selected conditions.

Moving the cursor over the plot creates a crosshair.

If the hover tool is enabled (see description below) moving over a data point will produce a tooltip with some basic information about the datapoint (compound, symmetry, full uniaxial magneto elastic parameter, volumetric magneto elastic parameter, internal field, applied field response).

Several selection tools can be used (see below) to pick data points. The default one is Tap. After selecting a data point all other data points will become semi-transparent. To discard selection, click anywhere in the plot area without datapoints. Selected data points are added to the table below the plot area which contains additional information about the material. See TABLE section for more details.

This 'selection' does not affect what is exported to the file when the 'Export data' button is pressed, it is used purely to populate the table below the plot area.

Immediately to the right of the plot area are additional tools to interact with the plot. Tools can be enabled and disabled with a mouse click. Enabled tools have a blue line next to them.
List of tools from top to bottom:

Pan - move plot area by clicking and dragging. Initial state - enabled.

Lasso Select - draw arbitrary shape inside plot area, all data points within will be selected. Initial state - disabled.

Box Zoom - draw a rectangle inside the plot area, the plot will be zoomed in to this rectangle. Initial state - disabled.

Box Select - draw a rectangle inside the plot area, all data points within will be selected. Initial state - disabled.

Wheel Zoom - allows zooming in and out by hovering the mouse pointer over the plot area and rotating the mouse wheel. Zoom location is determined by the position of the cursor. Initial state - enabled.

Tap - when enabled clicking on the datapoint will select it. Several data points can be selected if 'SHIFT' button is held down on the keyboard during mouse click. Initial state - enabled.

Reset - on click normalizes axis values (to use after zooming or switching axis).

Hover - first hover tool when enabled shows tooltip for black data points. Initial state - disabled.

Hover - second hover tool when enabled shows tooltip for orange data points. Initial state - enabled.

TABLE

Information on selected data points is automatically displayed in a table below the plot. This table has a pre-determined set of headers to give only the most useful information about compounds. Additional information is available in a .csv file that can be exported via the 'Export Data' button.

Click on any header to sort the whole table according to it. The first click results in numerically/alphabetically increasing ordering, the second - decreasing. The width of the table columns can be adjusted.
